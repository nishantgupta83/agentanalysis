Pipeline

  1. Parse local Codex and Claude JSONL logs
  2. Normalize usage events and tool-call rows
  3. Aggregate daily session, project, model, and source tables
  4. Read the dashboard through `GET /api/dashboard_data.php?days=N`

Security Boundary

  • Browser reads only JSON from PHP endpoints
  • Supabase service-role key stays server-side
  • Raw secrets belong in `.env` or `web/api/config.local.php` only
  • Public repo ships placeholders and sample payloads only

Endpoint Summary

Endpoint Purpose
GET /api/dashboard_data.php?days=N Returns aggregated dashboard payload plus row collections for drilldowns
GET /api/views.php Returns the page-view counter used by the public frontend
